what this mini-web covers

from switchboards to spine-leaf, from arp to bgp. this is the practical networking guide i wish i had on day one.

  • history: early telephony, arpanet, ethernet, tcp/ip, commercial internet, clos fabrics.
  • modern architecture: spine-leaf, ecmp, hashing, mtu, buffers, telemetry.
  • internet infrastructure: ases, ixps, peering, transit, rpki, anycast.
  • osi vs tcp/ip: mapping and layer-by-layer tools.
  • troubleshooting: commands, when to use them, how to read results.
  • arp and dns: deeper dives with captures and examples.

how to use this

start with history if you like stories. if you are debugging a live issue, jump straight to troubleshooting, then the arp/dns deep dives.

diagram note

all diagrams are inline svgs so you can copy, paste, and remix. no external deps.
